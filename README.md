# MealMind Monorepo

Turborepo workspace for the MealMind AI-assisted weekly meal planner demo. The scaffold includes:

- `apps/web`: Next.js App Router application for the demo UI
- `packages/api`: tRPC routers and shared contracts
- `packages/db`: Prisma client and migrations
- `packages/ai`: Prompt templates and AI orchestration helpers
- `packages/shared`: Cross-cutting utilities
- `tests/`: Playwright and contract test harnesses

Follow `specs/002-project-mealmind-ai/quickstart.md` for environment setup and execution details.
